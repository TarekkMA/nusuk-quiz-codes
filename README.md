# Code Viewer

A simple, mobile-friendly web application for displaying and managing quiz codes with a clean, elegant design.

## Features

- **Random Code Display**: Shows one code at a time from a curated list
- **Copy to Clipboard**: Easily copy any code with a single click
- **Progress Tracking**: Keeps track of used codes via localStorage
- **Mobile-Friendly**: Fully responsive design that works on all devices
- **Beautiful UI**: Clean white/gold/maroon color scheme with smooth animations

## How to Use

1. **Copy**: Click the "Copy" button or click on the code itself to copy it to your clipboard
2. **Next & Save**: Click "Next & Save" to:
   - Save the current code as "used"
   - Display a new random unused code
3. **Progress**: View how many codes remain in your list

## Technologies

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables and responsive design
- **Vanilla JavaScript**: No frameworks, pure JavaScript for functionality

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/TarekkMA/nusuk-quiz-codes.git
   ```

2. Navigate to the project directory:
   ```bash
   cd nusuk-quiz-codes
   ```

3. Open `index.html` in your browser or use a local server:
   ```bash
   # Python 3
   python3 -m http.server 8000

   # Node.js
   npx serve
   ```

4. Open your browser to `http://localhost:8000`
